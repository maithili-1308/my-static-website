# To Create Static Website 
## STEP:1
Launch EC2 instance.
## STEP:2
Install Nginx server.
## STEP:3
Create files - index.html, style.css.
## STEP:4
Deploy the static website by creating deploy.sh file.
## STEP:5
Configure Nginx to Serve the Static Site.
